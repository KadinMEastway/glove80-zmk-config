# MoErgo Glove80 Custom Configuration for ZMK

![MoErgo Logo](moergo_logo.png)

This repo was created from the official ZMK configuration of the MoErgo Glove80 wireless split contoured keyboard.

## Setup
To locate the firmware files and flash your Glove80 from this repo...<br>
2. Click "Actions" in the main navigation, and in the left navigation click the "Build" link.<br>
3. Select the desired workflow run in the centre area of the page (based on date and time of the build you wish to use). You can also start a new build from this page by clicking the "Run workflow" button.<br>
4. After clicking the desired workflow run, you should be presented with a section at the bottom of the page called "Artifacts". This section contains the results of your build, in a file called "glove80.uf2"<br>
5. Download the glove80.uf2<br>
6. Flash the firmware to Glove80 according to the user documentation on the official Glove80 Glove80 Support website (linked below)<br>
<br>
Your keyboard is now ready to use.<br>
