# MoErgo Glove80 Custom Configuration for ZMK

![MoErgo Logo](moergo_logo.png)

This repo was created from the official ZMK configuration of the MoErgo Glove80 wireless split contoured keyboard.

## Setup
To locate the firmware files and flash your Glove80 from this repo...<br>
1. Click "Actions" in the main navigation, and in the left navigation click the "Build" link.<br>
2. Select the desired workflow run. You can also start a new build from this page by clicking "Run workflow".<br>
3. Under the "Artifacts" section, locate a firmware file called "glove80.uf2"<br>
4. Download and flash the firmware to your Glove80. See official Glove80 Support website for instructions (linked below)<br>
<br>
Your keyboard is now ready to use.<br>
